# Examples.AspNetMvcCode.CodeUtility

La libreria è compilata per .netstandard 2.0 per essere utilizzabile anche su altre solution con target >= .NET framework 4.6




Questo progetto contiene:

- Estensioni per la verifica o manipolazione di dati all'interno della solution, in modo da eseguire queste operazioni in modo conciso  e leggibile.
Molti metodi sono adibiti alla conversione di valori da DateTime a string e viceversa.
- Convertitore di formato da utilizzare per la libreria moment.js 
- Patterns Regex più comuni


Quasi tutti i metodi e estensioni hanno un test corrispondente sul progetto
> Examples.AspNetMvcCode.CodeUtility.Tests
